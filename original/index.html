REdiscoverTE

Author: Haiyin Chen

Date: 2019-09

The REdiscoverTEpaper package provides methods and data associated with the paper Transposable Element Expression In Tumors Is Associated With Immune Infiltration And Increased Antigenicity by Kong Y, ..., Chen-Harris H (2019).

Contents:

  • R package that enables reproducibility of main figures

    Download location: http://research-pub.gene.com/REdiscoverTEpaper/data

  • REdiscoverTE software for whole transcriptome quantification of human RNAseq data

    Download location: http://research-pub.gene.com/REdiscoverTEpaper/software
